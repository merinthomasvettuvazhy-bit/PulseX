PulseX 🚨

Smart AI-Powered Emergency Safety Web Application
PulseX is an advanced emergency response and personal safety web application developed to provide immediate assistance during dangerous or critical situations. The application combines AI-powered voice commands, real-time location tracking, emergency communication, and safety automation into a single platform.
The main objective of PulseX is to help users quickly contact emergency services, share their live location, and alert trusted contacts during emergencies such as accidents, unsafe situations, medical emergencies, or threats.

PulseX is specially designed for:
👩 Women Safety
👧 Child Safety
🚑 Accident & Medical Emergencies

🌟 Key Features:
* 🎤 AI Voice Command System

Users can activate voice recognition and give emergency commands such as:
“Call Police”
“SOS”
“Share Location”
“Find Police”

The system automatically processes the command and performs the required emergency action without manual interaction.


* 🚨 Smart SOS Emergency Activation

The SOS system instantly:
Plays emergency alarm sound
Vibrates the device
Displays emergency alert screen
Calls emergency helpline numbers
Sends emergency data to backend server

This helps users react quickly during panic situations.

* 📍 Live Location Tracking

PulseX accesses the user’s GPS location and allows users to:
Share real-time location
Send Google Maps link through WhatsApp
Help emergency responders locate the victim faster

* 🚔 Nearby Police Station Finder

Using OpenStreetMap Overpass API, the application finds nearby:
Police stations
Emergency support centers
The system also shows:
Distance from user
Contact information
Direct Google Maps navigation

* 👥 Emergency Contact Management

Users can:

Add custom emergency contacts
Save trusted contacts locally
Quickly call contacts during emergencies
Remove or edit saved contacts

The contacts are stored using browser local storage.

* 🌐 Universal SOS Sharing

PulseX can generate and share an SOS emergency message with:

Current emergency status
Live location
Emergency help request

This can be instantly shared through WhatsApp.
